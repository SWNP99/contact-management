# Contact Management App

A **Contact Management CRUD application** built with **Django** and **HTMX**, demonstrating server-driven UI updates, live search, and Bootstrap-styled forms and tables. The app uses **PostgreSQL** as the database and showcases a beginner-friendly full-stack project.

---

## Features

- Add, edit, and delete contacts dynamically using **HTMX**.
- Live search/filter contacts as you type.
- Server-driven updates without writing custom JavaScript.
- Simple and clean Django backend architecture.
- CSRF-safe HTMX requests.

---

## Tech Stack

- **Backend:** Django 5.x
- **Frontend:** HTMX 2.x + Bootstrap 5
- **Database:** PostgreSQL
- **Python:** 3.13+
